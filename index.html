Entendendo a comunicação client x server

O que são clients?

(Voltar)

É um software ou dispositivo que solicita e recebe serviços de um servidor. Por exemplo, um navegador de internet
(como o Chrome ou Firefox) é um cliente que solicita páginas web de servidores para exibi-las aos usuários.

O que são servers?

(Voltar)

São sistemas, dispositivos ou programas que fornecem serviços, recursos, ou dados para outros sistemas,
dispositivos ou programas chamados de "clients" (clientes).

Linguagem de programação

(Voltar)

Um conjunto de regras, símbolos e sintaxe que permite aos programadores escreverem instruções que um
computador pode entender e executar. Essas instruções podem ser usadas para criar softwares, aplicativos, scripts, sites, e muito mais.

Introdução ao HTML na prática

Ferramentas utilizadas

(Voltar)

Editores de Código: Visual Studio Code, Sublime Text, Atom: Usados para escrever e editar HTML.
Ferramentas de Inspeção e Depuração: DevTools do Chrome, Firefox Developer Tools: Integradas aos navegadores para inspecionar e corrigir HTML, CSS e JavaScript.
Pré-processadores de CSS: Sass, LESS: Tornam o CSS mais dinâmico e fácil de manter.
Bibliotecas e Frameworks: Bootstrap, Tailwind CSS: Facilitam o design responsivo e estilização de páginas web.
Gerenciadores de Pacotes: npm, Yarn: Usados para instalar e gerenciar bibliotecas e ferramentas de desenvolvimento.
Ferramentas de Automação: Gulp, Webpack: Automatizam tarefas como minificação de arquivos e compilação de CSS.
Controle de Versão: Git, GitHub: Gerenciam o histórico de mudanças no código e facilitam a colaboração.
Testes e Acessibilidade: Lighthouse, Wave: Avaliam desempenho e acessibilidade de páginas web.

Estrutura básica do HTML

(Voltar)

A estrutura básica de um documento HTML começa com uma declaração do tipo de documento para definir a versão do HTML.
Segue-se o elemento raiz, que encapsula todo o conteúdo da página. Dentro dele, há duas seções principais: o cabeçalho e o corpo.
O cabeçalho inclui metadados, como o título da página, a codificação de caracteres, links para folhas de estilo, e outras informações
que não são exibidas diretamente na página. O corpo contém todo o conteúdo visível, como textos, títulos, parágrafos, imagens, links,
e outros elementos de mídia. Essa estrutura estabelece a base para a organização e a apresentação do conteúdo em uma página web.

Atributo das tags

(Voltar)

Atributos em HTML são usados para fornecer informações adicionais aos elementos. Eles são inseridos dentro da tag de abertura
de um elemento e consistem em um nome de atributo e um valor associado, onde o nome especifica o tipo de informação e o valor
define essa informação. Existem atributos globais que podem ser aplicados a praticamente qualquer elemento, permitindo a personalização
e identificação dos elementos na página. Além disso, alguns atributos são específicos para determinados elementos e controlam funções
ou comportamentos particulares desses elementos. Alguns atributos funcionam como booleanos, onde sua simples presença em um elemento ativa
uma função específica sem a necessidade de um valor explícito. Os atributos são essenciais para configurar as características e comportamentos dos
elementos HTML, influenciando a forma como eles são exibidos e interagem com o usuário.
